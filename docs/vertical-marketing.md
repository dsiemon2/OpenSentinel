# OpenSentinel Vertical Marketing Positioning

## 1. Financial Analysts & Traders

OpenSentinel is the only self-hosted AI assistant with built-in finance integrations:

- **Real-time market data** via Finnhub (stock quotes, company news, earnings, sentiment)
- **Economic indicators** via FRED (GDP, inflation, unemployment, interest rates)
- **Crypto trading** via Coinbase and Binance APIs with safety limits (max trade size, daily spend caps, per-hour rate limits)
- **DeFi analytics** via DeFiLlama
- **On-chain analytics** via Etherscan and Alchemy (wallet tracking, transaction history, token balances)
- **Alpha Vantage** for historical stock data and technical indicators

**Key differentiator:** No competitor offers ANY of these. OpenClaw has zero finance integrations.

### Use Case Examples

- "Check AAPL earnings and compare to last quarter"
- "What's the current GDP growth rate?"
- "Buy 0.1 ETH on Coinbase" (with confirmation prompts and safety limits)
- "Track whale wallet 0x... for large transactions"

## 2. OSINT Analysts & Investigators

OpenSentinel has a dedicated OSINT module with public records access:

- **FEC** (Federal Election Commission) -- campaign contributions, donor searches, committee filings
- **SEC EDGAR** -- company filings (10-K, 10-Q, 8-K), insider trading, corporate ownership
- **IRS 990** -- nonprofit tax returns, executive compensation
- **USASpending** -- federal contract and grant data
- **OpenCorporates** -- global company registry data, officers, filings
- **Neo4j graph database** -- entity relationship mapping, network visualization
- **Graph Explorer** -- visual web dashboard for exploring entity connections

**Key differentiator:** No other AI assistant has ANY public records integrations. This is a unique vertical.

### Use Case Examples

- "Who donated to [politician] in 2024?"
- "Show me SEC insider trading filings for Tesla executives"
- "Map the corporate network of [company] and its subsidiaries"
- "What federal contracts did [company] receive last year?"

## 3. Compliance Teams & Security Professionals

OpenSentinel is the only assistant with SOC 2-ready security controls:

- Full audit trail (every login, tool use, shell command logged with HMAC signatures)
- GDPR compliance tools (data export, right to deletion)
- 2FA/TOTP authentication
- Encrypted vault for API keys and secrets
- Role-based access control
- Prompt injection detection
- Tool sandboxing
- Prometheus metrics for monitoring

### Use Case Examples

- "Export all audit logs for user X between Jan 1 and Mar 1"
- "Show me all shell commands executed in the last 24 hours"
- "Generate GDPR data export for user Y"

OpenSentinel as an operational assistant:
