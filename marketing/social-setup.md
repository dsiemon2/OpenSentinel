# Social Accounts Setup Guide

## Required Accounts (Before Marketing Launch)

### 1. X/Twitter — @OpenSentinelAI
- Create account at twitter.com
- Set profile photo (use the og-image.svg shield logo)
- Bio: "Self-hosted AI assistant | 9 LLM providers, 124 tools, 7 channels | MIT licensed | Built with TypeScript + Bun"
- Link: https://opensentinel.ai
- Pin the launch thread (from launch-content.md Tweet 1)

### 2. Discord Server — OpenSentinel Community
- Create server at discord.com
- Channels:
  - #announcements (read-only)
  - #general
  - #support
  - #feature-requests
  - #show-and-tell (share setups)
  - #development (contributor discussion)
- Set invite link in website footer and README
- Add OpenSentinel bot to the server for demo

### 3. GitHub Discussions
- Already enabled at github.com/dsiemon2/OpenSentinel/discussions
- Post the welcome message from launch-content.md
- Set up categories: Q&A, Ideas, Show and Tell, General

## Post-Launch Accounts (Phase 2)

### 4. YouTube — OpenSentinel
- Create channel for demo videos and tutorials
- First video: "Getting Started with OpenSentinel in 5 Minutes"
- Use og-image.svg as channel art base

### 5. LinkedIn Company Page
- Create at linkedin.com/company/opensentinel
- For enterprise marketing and recruiter outreach
