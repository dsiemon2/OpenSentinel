# AI Competitor Tracker

Monitor competitor products, pricing, hiring, and strategy.

## What it does

- Researches competitors across web, GitHub, social media, and job boards
- Detects product launches and feature changes
- Analyzes hiring patterns to predict strategic direction
- Tracks pricing model changes
- Generates competitive landscape reports
- Stores intel history for trend analysis

## Quick start

```bash
cd templates/ai-competitor-tracker
bun install
CLAUDE_API_KEY=sk-ant-... bun run start
```

## Extend it

- Schedule weekly competitive intelligence reports
- Send alerts on major competitor moves
- Track competitor GitHub repos for release activity
- Build a competitive intelligence dashboard
