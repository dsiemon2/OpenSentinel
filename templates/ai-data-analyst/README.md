# AI Data Analyst

Analyze CSV/JSON datasets, find insights, detect anomalies, and generate reports.

## What it does

- Profiles datasets (column types, distributions, missing values)
- Identifies patterns, correlations, and outliers
- Answers natural language questions about your data
- Generates actionable recommendations from findings

## Quick start

```bash
cd templates/ai-data-analyst
bun install
CLAUDE_API_KEY=sk-ant-... bun run start
```

## Extend it

- Connect to databases for live query analysis
- Schedule daily reports on key metrics
- Export charts and visualizations
- Build a self-service analytics chatbot
