# AI Real Estate Analyst

Research properties, analyze markets, estimate ROI, and compare investment opportunities.

## What it does

- Analyzes local market conditions (median price, trends, rental yields)
- Evaluates individual properties with investment metrics (cap rate, cash-on-cash, gross yield)
- Finds comparable recent sales
- Identifies pros and cons per property
- Generates side-by-side comparison reports
- Stores analysis history for portfolio tracking

## Quick start

```bash
cd templates/ai-real-estate-analyst
bun install
CLAUDE_API_KEY=sk-ant-... bun run start
```

## Extend it
