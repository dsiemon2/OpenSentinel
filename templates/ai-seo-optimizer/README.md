# AI SEO Optimizer

Audit pages, research keywords, and generate optimized content outlines.

## What it does

- Comprehensive page audit (technical, content, meta, performance)
- Keyword research with related terms, questions, and long-tail variations
- Generates optimized title tags and meta descriptions
- Creates SEO-focused content outlines for writers
- Scores pages 0-100 with specific fix recommendations

## Quick start

```bash
cd templates/ai-seo-optimizer
bun install
CLAUDE_API_KEY=sk-ant-... bun run start
```

## Extend it

- Schedule recurring audits for your site pages
- Track ranking changes over time
- Generate content briefs for your writing team
- Integrate with Google Search Console for real ranking data
