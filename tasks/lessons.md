# Lessons Learned

## Twitter/X Outreach
- **NEVER blast multiple promotional tweets in rapid succession.** Space them 15-30 minutes apart minimum.
- The pending-tweets.md file literally said "spaced out" and we ignored it.
- New accounts with low followers get flagged faster for repetitive promotional content.
- Always ask the user about pacing before sending a batch — don't just fire them all off.
- Use background tasks or remind the user to come back for the next batch.
